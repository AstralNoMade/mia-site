# Сайт РОО «Медицинский Инвестиционный Альянс» (МИА)

Лендинг по [PRD v1.1](../prd.md), собранный на Next.js 14 + Tailwind CSS + Framer Motion.

## Быстрый старт

```powershell
# В этой папке (C:\dev\mia-site):
npm run dev
```

Откройте http://localhost:3000

## Команды

| Команда | Действие |
|---|---|
| `npm run dev` | dev-сервер с hot-reload на :3000 |
| `npm run build` | production-сборка в `.next/` |
| `npm start` | запуск production-сборки |
| `npm run lint` | ESLint |

## Структура

```
mia-site/
├── app/
│   ├── globals.css     ← брендовые CSS-переменные из PRD §2.1
│   ├── layout.tsx      ← root layout, шрифты (Unbounded, PT Serif, JetBrains Mono)
│   └── page.tsx        ← композиция 14 секций
├── components/
│   ├── header.tsx      ← sticky-хедер с якорной навигацией
│   ├── footer.tsx
│   └── sections/       ← 14 секций лендинга (Hero, About, ... Contact)
├── public/logo/        ← 5 PNG логотипов от заказчика
├── tailwind.config.ts  ← цвета mia-* привязаны к CSS-переменным
└── package.json
```

## Бренд

Цвета, типографика и тон взяты строго из брендбука (PRD §2). Изменять без согласования
с автором визуального концепта — **Зейнеш Абжановой (+7 701 714 19 50)** — нельзя.

Что нужно от заказчика для продакшена:
- [ ] Векторный логотип (SVG/AI/EPS) — сейчас используются PNG 2884×2150
- [ ] Лицензия на шрифт **Geometr415 Blk BT** (сейчас замена — **Unbounded**)
- [ ] Реальные фото основателей (ч/б в стиле брендбука)
- [ ] Тексты Устава, Политики конфиденциальности
- [ ] Реальные QR-коды WhatsApp / Telegram (сейчас плейсхолдеры)

## Что реализовано (Этап 1 PRD §11)

- [x] Все 14 секций лендинга, статика
- [x] Адаптив 360 / 768 / 1024 / 1440 px
- [x] Якорная навигация и плавный скролл
- [x] 4-шаговый wizard анкеты с валидацией на клиенте
- [x] Хедер с переключателем языков (UI, без перевода)
- [x] Анимации появления на скролле (Framer Motion)
- [x] Логотипы в правильных цветах для каждого фона (брендбук §2.2)

## Что НЕ реализовано (требует Этап 2)

- Backend API и БД (PostgreSQL + Prisma)
- Реальная отправка анкеты (сейчас — `console.log` + UI «спасибо»)
- i18n переводы (`next-intl`) — есть только UI-переключатель
- Личный кабинет (`/cabinet/*`)
- Платежи (Kaspi/Halyk/Stripe)
- CMS для новостей / событий
- reCAPTCHA v3

## Important: OneDrive и Node.js

Проект был перенесён из исходной папки `C:\Users\leila\OneDrive\Рабочий стол\Мой AI\Проект сайта`
в `C:\dev\mia-site`, потому что OneDrive блокирует файлы `node_modules` во время сборки
(ошибка `EBUSY: resource busy or locked`). Это известная проблема — все Node.js-проекты
нужно держать вне синхронизируемых папок.

В исходной OneDrive-папке оставлена копия исходников + указатель на эту локацию.

## Security note

Next.js 14.2 содержит security-advisory от 11.12.2025 (см.
https://nextjs.org/blog/security-update-2025-12-11). Для статического лендинга без
auth/API-роутов риск минимальный. Перед запуском в продакшен следует обновиться
на Next.js 15.
